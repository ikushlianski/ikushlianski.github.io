Cat Clicker

GitHub

Purpose:

Learn the Model-View-Controller concept.

Description:

You click the cat and its image is displayed. After clicking on the cat's image the number of clicks next to the corresponding cat increases by 1. You can switch between cats and the number of clicks is kept in place.

Technologies used:

HTML, CSS, Flexbox, Javascript, jQuery and a bit of Bootstrap

Timing:

1 evening

Conclusion:

This is what led me to learn Angular 2, which relies on MVC. Of course there's no sense to use Angular for such small apps.
